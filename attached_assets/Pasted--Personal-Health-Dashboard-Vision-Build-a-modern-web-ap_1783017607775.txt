# Personal Health Dashboard

## Vision

Build a modern web application that transforms raw pathology reports into a living health record. Instead of browsing dozens of PDFs, users should have a beautiful dashboard that shows their current health, trends over time, abnormal values, and improvements.

The application should work entirely from the canonical parsed JSON, making it independent of the laboratory (Orange Health, Tata 1mg, etc.).

---
